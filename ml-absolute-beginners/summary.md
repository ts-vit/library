# Выжимка: Машинное обучение для абсолютных новичков (Оливер Теобальд)

Идеальный вводный курс (2024), объясняющий основные алгоритмы ML простыми словами.

### Ключевые темы:
1.  **Основы:** Что такое ML, тренировочные и тестовые данные.
2.  **Алгоритмы:** Линейная и логистическая регрессия, метод K-ближайших соседей, деревья решений, случайный лес.
3.  **Подготовка данных:** Очистка, нормализация, стандартизация и обработка пропусков.
4.  **Практика:** Построение базовой модели на Python с использованием библиотек Pandas и Scikit-Learn.

### Для кого эта книга:
Для тех, кто только начинает свой путь в Data Science и хочет получить системное представление об области.
