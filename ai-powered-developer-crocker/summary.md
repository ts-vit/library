# Выжимка: Разработка с ИИ (Нэйтан Крокер)

Практическое руководство (2025) по внедрению ChatGPT, Copilot и других инструментов в ежедневный рабочий цикл программиста.

### Ключевые темы:
1.  **Проектирование и архитектура:** Использование ChatGPT для создания ITAM-систем, документирование архитектуры и проектирование модели предметной области.
2.  **Реализация:** Написание кода с GitHub Copilot, использование паттернов Factory и Builder, работа с гексагональной архитектурой.
3.  **Тестирование:** Автоматизация юнит-тестов, интеграционное и поведенческое тестирование с помощью LLM. Анализ и объяснение чужого кода.
4.  **Развертывание и инфраструктура:** Настройка Docker, Terraform и CI/CD пайплайнов с помощью ИИ-ассистентов.
5.  **Безопасность:** Моделирование угроз с ChatGPT и поиск уязвимостей в структуре приложения.
6.  **Локальный запуск:** Работа с GPT без интернета (Llama 2, GPT-4All).

### Для кого эта книга:
Для Fullstack и Backend разработчиков, которые хотят ускорить свой рабочий процесс и улучшить качество кода.
