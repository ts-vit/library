# Выжимка: Грокаем глубокое обучение (Эндрю Траск)

Визуальное руководство по основам нейронных сетей, позволяющее понять математику и логику "под капотом" без использования сложных библиотек.

### Ключевые концепции:
1.  **Прогнозирование:** Как нейронные сети делают предсказания через взвешенные суммы.
2.  **Градиентный спуск:** Оптимизация весов и минимизация ошибок (метод "холодно-горячо").
3.  **Обратное распространение (Backpropagation):** Как сигнал об ошибке проходит сквозь слои сети.
4.  **Регуляризация:** Борьба с переобучением через Dropout и другие методы.
5.  **Обработка естественного языка (NLP):** Работа с векторами слов и предсказание следующего слова.

### Для кого эта книга:
Для новичков, которые хотят понять фундаментальные принципы работы ИИ, прежде чем переходить к готовым фреймворкам.
