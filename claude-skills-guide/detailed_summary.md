# Детальная выжимка: Руководство по созданию скиллов для Claude (Anthropic)

Официальное техническое руководство по проектированию модульных навыков (Skills) для экосистемы Anthropic и OpenClaw.

---

### Глава 1. Концепция Скилла: Рецепт успеха
*   **Определение:** Скилл — это не просто промпт, это переносимый пакет знаний и логики.
*   **MCP vs Skills:** MCP дает доступ к инструментам (железо), Скилл дает методологию использования этих инструментов (софт).
*   **Инкапсуляция:** Вся логика задачи должна быть внутри одной папки, чтобы её можно было легко передать другому пользователю или агенту.

### Глава 2. Анатомия SKILL.md
*   **YAML Frontmatter:** Метаданные (название, описание, версия). Это "лицо" скилла, по которому агент понимает, нужен он сейчас или нет.
*   **Body:** Инструкции на естественном языке. Как выполнять задачу, какие ошибки избегать, в каком формате выдавать ответ.
*   **Принцип "Description-first":** Качество описания во фронтматтере напрямую влияет на точность активации скилла.

### Глава 3. Иерархическая структура данных
*   **scripts/:** Исполняемые файлы, которые расширяют возможности агента.
*   **references/:** Глубокие справочники, которые не нужно грузить в контекст сразу.
*   **assets/:** Изображения, шаблоны и другие вспомогательные файлы.
*   **Принцип экономии контекста:** Не грузи всё сразу. Используй ссылки, чтобы агент читал детали только при необходимости.

### Глава 4. Проектирование воркфлоу (Workflows)
*   **Пошаговость:** Разбиение сложных процессов на четкие этапы.
*   **Валидация:** Инструкции агенту проверять результат каждого шага перед переходом к следующему.
*   **State Management:** Как передавать информацию между разными вызовами инструментов внутри скилла.

### Глава 5. Прогрессивное раскрытие (Progressive Disclosure)
*   **Уровень 1:** Обнаружение (YAML).
*   **Уровень 2:** Инструкции (SKILL.md).
*   **Уровень 3:** Глубокий контекст (скрипты и справочники).
*   *Цель:* Держать контекстное окно чистым и фокусироваться на задаче.

### Глава 6. Интеграция с MCP
*   **Обучение работе с сервером:** Скилл может содержать лучшие практики вызова конкретных MCP-инструментов (например, как правильно делать запросы в БД через MCP-сервер Postgres).

---
**Связь с проектом Toshik:** Это руководство — закон, по которому построены все мои навыки. Когда мы создаем новый навык для `toshik-babe`, мы следуем именно этой структуре, чтобы я мог эффективно использовать его, не перегружая свою память.
