# Выжимка: Введение в автоматизированное ML (AutoML)

Эта книга представляет собой глубокое погружение в мир автоматизации машинного обучения. Основной упор сделан на **метаобучение** — концепцию «обучения тому, как обучаться».

### Ключевые темы:
1.  **Метаобучение (Meta-Learning):** Использование опыта решения прошлых задач для ускорения обучения на новых. Вместо обучения с нуля, система использует «индуктивное смещение» (inductive bias).
2.  **Алгоритмы и архитектуры:**
    *   **MAML (Model-Agnostic Meta-Learning):** Поиск такой инициализации параметров, которая легко адаптируется к новым задачам.
    *   **Matching Networks & Prototypical Networks:** Использование метрик расстояния и компонентов памяти для обучения на малых данных (few-shot learning).
    *   **MANN (Memory-Augmented Neural Networks):** Нейронные сети с внешней памятью (например, машины Тьюринга).
3.  **Применение AutoML:**
    *   Автоматический выбор алгоритмов и гиперпараметров.
    *   Активное обучение (Active Learning).
    *   Оценка плотности и рекомендательные системы.
4.  **Философия:** Накопление «истории обучения» для создания систем, которые постоянно совершенствуются.

### Для кого эта книга:
Для специалистов по Data Science, ML-инженеров и исследователей, которые хотят выйти за рамки ручного подбора моделей и строить самообучающиеся системы.
